# Scripts vs Codegen
1. Scripts are for all one off operations that you only run when you need something to happen
1. Codegen
    1. Codegen tools are tools/frameworks/scripts that are used to generate code, which can be a PART of the build process that ALWAYS run to generate code needed in order for the codebase to run normally.
    1. Codegen artifacts are the output of your codegen tools, these can be generated code or generated docs or something related.
